Proje 1 - Hesap Makinesi


Bu projede temel dört işlemi yapabilen basit bir web tabanlı hesap makinesi geliştirdim. HTML, CSS ve JavaScript kullanarak oluşturduğum bu uygulama, kullanıcıların tarayıcı üzerinden kolayca işlem yapabilmesini sağlıyor.
Hem tasarımı sade tuttum hem de fonksiyonel olmasına özen gösterdim. Bu projeyle birlikte JavaScript pratiği yapma ve etkileşimli arayüzler oluşturma konusunda kendimi geliştirdim.

Proje 2 - Yapılacaklar listesi

Bu projede, JavaScript öğrenme sürecimde temel mantığı pekiştirmek amacıyla bir yapılacaklar listesi uygulaması geliştirdim. Kullanıcı, görev ekleyebiliyor, eklediği görevleri tamamlandı olarak işaretleyebiliyor veya silebiliyor.
Amacım, hem DOM manipülasyonu pratiği yapmak hem de küçük ama işlevsel bir uygulama ortaya koymaktı. Tasarımı sade tuttum, temel HTML ve CSS kullandım. JavaScript tarafında ise olay dinleyicileri, element oluşturma ve class kontrolü gibi temel konuları uyguladım.
